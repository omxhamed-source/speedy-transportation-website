# Speedy Transportation LLC Website

Open `index.html` in a browser to view the site.

The homepage uses:

- `speedy-logo.png` for the brand logo
- `hero-video.mp4` for the cinematic trucking hero background

## Netlify Forms

The forms are prepared for Netlify Forms:

- `driver-application`
- `freight-request`
- `chat-message`

After deploying on Netlify, open the site dashboard, go to **Forms**, and add email notifications for the address where applications and messages should go.

Recommended notification email: `dispatch@speedytransportationllc.com`.
